🏏 T20 World Cup 2026 – SQL Project

📌 Overview
This project is a complete SQL-based analysis of a simulated T20 World Cup 2026 tournament. It includes database creation, data insertion, and analytical queries to extract meaningful insights from match and player data.

🗂️ Database Schema
	Tables Included:
	teams
	players
	venues
	matches
	batting_scorecards
	bowling_scorecards
	fielding_stats
	tournament_batting_summary
	tournament_bowling_summary
	
⚙️ Setup Instructions
	1. Create Database
	2. Create Tables
	3. Insert Data
	4. Execute Queries

Compatible with MySQL (minor adjustments may be needed for other databases).

📊 SQL Queries Included
	1. Top Run Scorers
		* Total runs, strike rate, average
	2. Top Wicket Takers
		Wickets, economy, bowling strike rate
	3. Highest Strike Rate (Min 50 balls)
	4. Best Economy Bowlers (Min 16 overs)
	5. Match Results
		* Winner, margin, player of the match
	6. Team Performance
		* Matches, wins, losses, win %
	7. Most Sixes Hit
	8. Highest Individual Scores
	9. Best Bowling Figures
	10. Semi-Finals & Final Results
	11. Century Scorers
	12. India Match Journey
	13. Venue Statistics
	14. Top Fielders
	15. Player of the Match Awards
	16. Toss Impact Analysis
	17. Group Stage Standings
	18. Final Match Batting Scorecard
	19. Final Match Bowling Scorecard
	20. Super 8 Results

🧠 Concepts Used
	Joins (INNER, LEFT)
	Aggregations (SUM, COUNT, AVG)
	CASE statements
	NULL handling (COALESCE, NULLIF)
	Group By & Ordering
	Relational Data Modeling
